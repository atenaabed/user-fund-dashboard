# user-fund-dashboard

This repository contains scripts to fetch, process, and analyze user account data for national funds.  
The data is collected from the DeBank platform and processed to calculate specific metrics, including budgets, profits, and other financial information. The results are exported in Excel format for reporting and further analysis.
